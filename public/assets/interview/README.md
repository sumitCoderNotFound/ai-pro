# Interview room assets (optional)

Drop these two files here to upgrade the candidate interview screen visuals.
Both are OPTIONAL — if a file is missing, the UI falls back gracefully
(a CSS gradient background and an animated SVG avatar).

1. interview-room-bg.jpg
   - A blurred, professional office / meeting-room background.
   - Landscape, at least 1600x900. It is blurred and darkened in the UI.

2. avatar-female-interviewer.png
   - A professional, semi-realistic female interviewer headshot.
   - Square (e.g. 600x600), transparent or plain background, head & shoulders.
   - Original artwork or a licensed/own image. Do not copy another platform's avatar.

No code changes are needed after adding the files; the screen picks them up
automatically. You can also set a per-workspace avatar photo URL in
Recruitment Settings, which overrides the default avatar file.

## Seated interviewer (recommended for the room look)
3. avatar-female-interviewer-seated.png
   - The interviewer SEATED, upper body, with a TRANSPARENT background (PNG cut-out).
   - She is composited on top of interview-room-bg.jpg, so the room image should
     contain NO person (room only).
   - Tip: take a seated photo and remove its background (e.g. remove.bg) to get a
     transparent PNG. Portrait/upper-body framing; feet need not be visible.
   - If this file is missing, the screen falls back to the centered avatar.
